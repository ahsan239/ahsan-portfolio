# **App Name**: AI Solutions Showcase

## Core Features:

- Project Overview & Discovery: Showcase featured AI projects on the homepage with high-level business impact, a value proposition, and social proof through a tech stack display.
- Detailed AI Case Studies: Provide dedicated project pages detailing the business problem, the high-level AI solution, key ROI metrics, and a collapsible technical deep-dive.
- Interactive AI Demo Tool: Offer an embedded, interactive playground within project case studies, allowing users to experience a limited version of the AI solution in real-time, leveraging an LLM for its functionality and protected by a backend proxy.
- Service Offerings & Consultation Booking: Feature a dedicated 'Hire Me' page outlining service offerings for recruiters and clients, complete with an integrated calendar booking tool (e.g., Cal.com) for scheduling discovery calls.
- Dynamic Content Management: Implement a Content Management System (Sanity.io or MDX) to easily add, edit, and publish new project case studies without requiring code changes.

## Style Guidelines:

- A dark theme is chosen to align with the preference of engineers and evoke a modern, tech-focused aesthetic. The background color is a deep, slightly bluish charcoal (#22262B), creating a clean and professional base. The primary color is a striking yet balanced tech blue (#7AA3E0), offering high legibility and conveying a sense of innovation for interactive elements and headlines. An analogous, vibrant electric blue (#00DFFF) serves as an accent, strategically highlighting AI-specific elements and critical calls to action to create a 'neon' tech feel.
- The 'Inter' sans-serif typeface is recommended for its highly legible, modern, and objective characteristics, suitable for both headlines and body text to maintain a clean tech aesthetic.
- Utilize minimalist, clean line-art or geometric icons that complement the 'Clean Tech' theme, focusing on clarity and functionality over decorative embellishments.
- Implement a minimalist design with generous whitespace to ensure readability and focus. Project listings should adopt a modern 'Bento Grid' layout, while case studies will feature structured content areas and collapsible sections for technical deep dives.
- Integrate subtle, premium interactions using Framer Motion, including tasteful fade-ins for content loading, smooth scrolling transitions, and understated hover effects to enhance the user experience without being distracting.
